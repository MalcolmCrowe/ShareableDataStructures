The Shared folder is the location for debugging and staged solutions of
almost all of the PyrrhoDB software. 

At the moment, the distribution includes the following 4 binaries
for Windows and Mono:

PyrrhoCmd.exe, PyrrhoSvr.exe, PyrrhoLink.dll, EmbeddedPyrrho.dll

The distribution also contains other compilable solutions (NB: C# 2017 required).

(ProfileViewer:) OSPProfile
(PyrrhoCmd:) OSPCmd, PyrrhoCmd
(PyrrhoSQL:) OSPSQL
RESTClient
Tpcc

The sources for restif are on github.com/MalcolmCrowe/restif.
If anything seems to be missing please email me: Malcolm.Crowe@uws.ac.uk
